# EasyConverter ffmpeg
Adds ffmpeg conversion to the context menu for easy and fast media file conversion.

How to use:

Make sure you have installed ffmpeg > https://ffmpeg.org/download.html

ffmpeg guide > https://www.geeksforgeeks.org/installation-guide/how-to-install-ffmpeg-on-windows



Download the source code
Drop it into any folder (the program will be stored here)
Run install.py
